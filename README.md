# 🛒 E-Commerce Sales Analysis

A Python-based data analysis project that dives into e-commerce sales data to uncover actionable business insights — from revenue trends and top-performing products to customer behavior and seasonal patterns.

---

## 📌 Project Overview

This project performs an end-to-end exploratory data analysis (EDA) on an e-commerce sales dataset. Using Python's data science ecosystem, it cleans raw transactional data, engineers useful features, and produces visualizations that help answer key business questions such as:

- Which products and categories drive the most revenue?
- How do sales fluctuate across months and seasons?
- What does the customer purchase behavior look like?
- Which regions or cities contribute the most to overall sales?

## 📊 Key Analyses Performed

- **Sales Trend Analysis** — Monthly and yearly revenue breakdowns
- **Product Performance** — Top-selling items and categories by revenue and volume
- **Customer Insights** — Purchase frequency, average order value, repeat buyers
- **Regional Analysis** — Sales distribution across cities/states
- **Profit Margin Analysis** — Identifying high-margin vs. low-margin products
- **Correlation Analysis** — Relationships between discount, quantity, and profit

## 🚀 Getting Started

### Prerequisites

Make sure you have Python 3.7+ installed. Then install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Running the Analysis

1. **Clone the repository**

```bash
git clone https://github.com/AADITYA-NAHAR-JAIN/E-Commerce-Sales-Analysis.git
cd E-Commerce-Sales-Analysis
```

2. **Navigate to the project folder**

```bash
cd "E-Commerce Sales Analysis"
```

3. **Launch Jupyter Notebook**

```bash
jupyter notebook
```

4. Open the notebook(s) and run all cells to reproduce the analysis.
